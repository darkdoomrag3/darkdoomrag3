## About

Application Security Engineer with a software engineering and DevSecOps background, experienced
securing web applications, APIs, cloud infrastructure, and AI-driven platforms.

I specialize in penetration testing, secure code review, vulnerability management, security
automation, CI/CD security, and secure SDLC practices, backed by 6+ years of software engineering
experience. Proven track record identifying critical vulnerabilities including RCE, command
injection, authorization flaws, business-logic vulnerabilities, race conditions, and OWASP Top 10
risks.

## Selected Impact

- Identified and helped remediate RCE, command injection, authorization flaws, business-logic
  vulnerabilities, race conditions, secret exposure, and OWASP Top 10 risks.
- Integrated Semgrep, Trivy, OWASP ZAP, GitLeaks, and custom security tooling into CI/CD
  pipelines and automated security gates.
- Reduced GCP cloud costs from about $600/day to about $100/day through API, database, and
  infrastructure optimization initiatives.
- Implemented Terraform-based security controls and GCP Cloud Armor protections, identifying
  infrastructure security weaknesses during deployment reviews.
- Owned application security testing on a platform serving 20M+ users, finding business-logic
  flaws, race conditions, broken access control, and API authorization bypasses.
